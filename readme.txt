1. Copy "sigcreator" to your Desktop

2. Run sigcreator and follow the steps on the interface. (Best to do after user is assigned both of their numbers)

3. Once you've filled out the prompts it will generate a new .htm file in the Public Drive named User'ssignature (Ex. liam.hoganson's signature)

4. On the user's laptop, open  the Outlook app and add their profile to the Outlok app.

4. copy the "copy.ps1" file to the users's machine and run it with Powershell. (Must be ran via Right click --> 'Run with Powershell')

5. Boot up Outlook and it will now be an option under signatures. Be sure to set the signature to: New Messages and Replies/Forwards
